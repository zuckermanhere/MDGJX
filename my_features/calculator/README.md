# 简易计算器

这是 `my_features` 目录下的第一个自定义功能——一个命令行简易计算器。

## 使用方法

### 构建
```bash
cd my_features/calculator
go build -o calc.exe .
```

### 运行
```bash
# 加法
go run . -a 10 -b 5 -op "+"
# 输出: 10 + 5 = 15

# 减法
go run . -a 10 -b 5 -op "-"
# 输出: 10 - 5 = 5

# 乘法
go run . -a 10 -b 5 -op "*"
# 输出: 10 * 5 = 50

# 除法
go run . -a 10 -b 5 -op "/"
# 输出: 10 / 5 = 2
```

## 参数说明
| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `-a` | float64 | 0 | 第一个数字 |
| `-b` | float64 | 0 | 第二个数字 |
| `-op` | string | `+` | 运算符：`+` `-` `*` `/` |
